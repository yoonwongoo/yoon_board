# 스프링 프레임워크를 이용한 게시판만들기
## 개발환경
* Window Pro 10  
* STS4, jdk11  
* oracle18c XE  
* tomcat 8.5
* mybatis3.5.3
## 학습목표
CRUD기술을 학습을 목표로 하는 개인 프로젝트.  
* 등록
* 검색 (유형별 검색)
* 조회  
* 수정  
* 삭제
